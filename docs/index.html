Exile Bot

@PathOfExileBot

@PathOfExileBot is a Telegram Bot aimed at Path of Exile players. It's intended to be a helper and provide common functionalities to save time or work as a tool for player's Telegram groups. The code is up on Github, if you wanna check it out.

Commands

/about - Information about the bot

/greetings - Replies with one of the masters' greetings

/help - Shows a list of commands

/lab <difficulty> - Retrieves last lab layout for given <difficulty> from Poelab

/onsale - Shows a list of items on sale

/resources - Shows a list of helpful links

/safelevels - Level range with no experience penalty for <level>

/unique <name> - Retrieves information and wiki link of unique item <name>

/wiki <term> - Returns closest matching wiki link for given <term>

/wisdom - One of Izaro's many, many quotes
