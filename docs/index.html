Exile Bot

@PathOfExileBot

@PathOfExileBot - or Exile Bot - is a Telegram Bot aimed at Path of Exile players. It's intended to be a helper and provide common functionalities to save time or work as a tool for player's Telegram groups.

Commands

/about - Information about the bot

/greetings - Replies with one of the masters' greetings

/help - Shows a list of commands

/lab - Retrieves last lab layout for given difficulty from Poelab

/resources - Shows a list of helpful links

/unique - Retrieves information and wiki link of unique item name

/wisdom - One of Izaro's many, many quotes
